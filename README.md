# Active Directory Home Lab on Microsoft Azure

## Project Overview
This project simulates an enterprise Active Directory environment deployed on Microsoft Azure. It demonstrates core Identity and Access Management (IAM) skills including user provisioning, group management, organizational unit design, and access auditing using PowerShell.

## Technologies Used
- Microsoft Azure (Virtual Machine - Windows Server 2022)
- Active Directory Domain Services (AD DS)
- PowerShell
- Remote Desktop Protocol (RDP)

## What Was Built
- Deployed a Windows Server 2022 VM on Azure
- Promoted the server to a Domain Controller for the domain `carltonlab.local`
- Created Organizational Units (OUs) to simulate enterprise structure
- Created and managed user accounts (e.g., jsmith / John Smith)
- Created security groups including a custom IT-Staff group
- Assigned users to appropriate security groups
- Performed IAM audit to identify disabled/inactive accounts
